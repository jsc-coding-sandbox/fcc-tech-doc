Linux Commands

tar xclip cat wget mount

tar

Used to archive files. Can also be used to compress using a compression algorithm like GZip or 7zip, but does not do this by default.

  • https://man.archlinux.org/man/core/tar/tar.1.en

Here is an example of the tar command being used to archive and compress all JPG images from the current directory into a tar.gz file: tar -czf archive.tar.gz ./*.jpg

xclip

Allows us to move data into X selection (the clipboard on systems using an implementation of the X Window System).

  • https://man.archlinux.org/man/xclip.1

Here is an example of using xclip to put the contents of file "requirements.txt" into the clipboard: cat requirements.txt | xclip -selection clipboard

cat

Prints the contents of a file to stdout.

  • https://man.archlinux.org/man/cat.1

Here's an example of using cat to print the file "main.cpp" with line numbers: cat -n main.cpp

wget

Used to download pages from the internet.

  • https://man.archlinux.org/man/cat.1

Here is an example of using wget to download the web page "companysite.com/secrets.html", using the DNS server "9.9.9.9" (made up IP btw) instead of the default: wget --dns-servers=9.9.9.9 http://companysite.com/secrets.html

mount

Mounts a filesystem so that it can be explored in the host OS. Used to "connect" an internal drive which isn't part of the OS (i.e. your Windows dual boot), an external drive, a USB stick, or another storage device.

  • https://man.archlinux.org/man/mount.8

Here's an example of using mount to attach the drive & partition "/dev/sdc1" to the directory "/run/mount/externalhdd" (note that the directory must exist first!): mount /dev/sdc1 /run/mount/externalhdd
